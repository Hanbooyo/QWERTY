· Java 11  
· Spring  
· Maven  
· JSP  
· Mybatis  
· WebSocket

· RDBMS (MS SQL, Oracle)

· 경기일정 (sports 무료 API, youtube API 활용)  
· 실시간 익명채팅 (Websocket 활용)  
· 게시판 (Spring MVC)  
· 회원가입/회원정보관리
